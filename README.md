# Backend API Retinoid

## Intruduction
### Setup and Installation

```diff 
  $ git clone git@github.com:Adysa-Marketing/be-retinoid.git
  $ cd be-retinoid
```
- Setup .env file

- Installation and Running Project

```diff
  $ npm install
  $ npm run start
  $ nodemon
```

- format code automatically

```diff
  $ npm run format
```
